A modular benchmark for robot dexterity

HiveBoard combines a seven-cell press-fit base, interchangeable attachments, and a fixed protocol for evaluating manipulation interfaces across laboratories. Fabrication and simulation files are public.

01 / Benchmark

One base, interchangeable industrial tasks

Each attachment uses the same hexagonal footprint and seats in the base without fasteners. A laboratory can print the hardware, change the board configuration between trials, and run the same task definitions on different robot platforms.

The protocol evaluates the manipulation system as a whole: end-effector, control interface, perception, and operator input where applicable. It records task performance; it does not produce a single platform ranking.

04 / Protocol

Fixed procedure, task-level data

  1. 01

    Stabilize practice trials

    Record only after three consecutive familiarization trials complete within 20% of each other, or reach the timeout.

  2. 02

    Run five recorded trials

    Use the same neutral start pose for each platform and reset the attachment after every trial.

  3. 03

    Log the defined fields

    Outcome, failure cause, completion time, attempts, regrasps, and stage reached for assembly tasks.

  4. 04

    Return one evaluation archive

    Include trials.csv, setup.jpg, and platform.md.
